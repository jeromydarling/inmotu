# Community intelligence briefs

Research briefs that inform how inmotu speaks to and builds for each grassroots
motorsports community. Used to drive sector selection at onboarding, terminology,
the ladder/progression models, and marketing copy.

Each community is a **sector** a user can pick at onboarding so the product
adapts its vocabulary and features to their world.

- [`bmx.md`](./bmx.md) — USA BMX racing (moto/main, NAG, Gold Cup, the Grands)
- [`drag.md`](./drag.md) — grassroots/sportsman drag racing (bracket, the tree, dial-in, Jr. Dragster)
- [`karting.md`](./karting.md) — sprint **and** dirt-oval karting (LO206, the sprint/dirt divide)

> Briefs verified against 2025–2026 sources. Treat dated specifics (schedules,
> payouts, exact win thresholds, class ET caps) as needing annual re-verification
> against the governing body's current rulebook before hard-coding.
